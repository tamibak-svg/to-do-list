עיקרון חשוב מאוד

כל feature חדש צריך לעבור 3 שאלות:

1. האם זה עובד בלוקל?
2. האם זה עובד עם RLS והרשאות?
3. האם זה עובד גם אחרי deploy?

אם אחת מהתשובות היא לא, לא ממשיכים הלאה.

הניסוח שאני מציע להכניס ל־PROJECT STATE / ROADMAP
Testing & Release Workflow

Before every production deploy:

test locally first

verify create / read / update / delete

verify user isolation and RLS

verify delete flows carefully

push only after local validation

verify production after deploy

only then continue to next feature
